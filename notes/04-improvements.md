# Improvements

## Рекомендуемые улучшения

1. Вынести секреты (`Wi-Fi`/`MQTT`) из скетча в отдельный локальный файл-конфиг, не коммитить в git.
2. Добавить таймаут в `connectWiFi()`, чтобы избежать бесконечного ожидания.
3. Добавить экспоненциальный backoff для MQTT reconnect вместо фиксированных 2 секунд.
4. Логировать статус подключения в Serial для диагностики.
5. Добавить OTA-обновление, чтобы не перепрошивать вручную по USB.

## Быстрый чек после прошивки

- Устройство подключилось к Wi-Fi.
- На брокере появился `ha/<nodeId>/status = online`.
- В Home Assistant появился switch через discovery.
- Команды `ON/OFF` реально переключают GPIO4 (вспышку).
