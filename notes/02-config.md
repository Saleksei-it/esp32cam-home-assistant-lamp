# Config

## Что заменить перед прошивкой

В файле `esp32cam_flash_ha.ino`:

- `WIFI_SSID`
- `WIFI_PASSWORD`
- `MQTT_HOST`
- `MQTT_PORT`
- `MQTT_USER`
- `MQTT_PASSWORD`

## Примечания по параметрам

- Если MQTT без авторизации, оставьте `MQTT_USER` пустой строкой (`""`), тогда используется ветка подключения без логина.
- `MQTT_HOST` лучше задавать статическим IP брокера или стабильным DNS-именем.
- Топики автоматически формируются на основе уникального `nodeId`, вручную их обычно менять не нужно.
